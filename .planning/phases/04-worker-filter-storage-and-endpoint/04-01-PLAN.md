Add filter storage and version tracking to SqlTask to hold filters from DynamicFilterSourceOperator.

1. Add field: `private final ConcurrentMap<String, TupleDomain<String>> dynamicFilters = new ConcurrentHashMap<>();`
   - Key: filterId (String), Value: TupleDomain<String> (filter constraint)

2. Add field: `private final AtomicLong outputsVersion = new AtomicLong(0);`
   - Tracks version for incremental fetching

3. Add method to store filter (called by DynamicFilterSourceOperator via consumer):
```java
public void addDynamicFilter(String filterId, TupleDomain<String> filter) {
    requireNonNull(filterId, "filterId is null");
    requireNonNull(filter, "filter is null");
    dynamicFilters.put(filterId, filter);
    outputsVersion.incrementAndGet();
}
```

4. Add method to get filters since version:
```java
public Map<String, TupleDomain<String>> getDynamicFiltersSince(long sinceVersion) {
    if (outputsVersion.get() <= sinceVersion) {
        return ImmutableMap.of();
    }
    return ImmutableMap.copyOf(dynamicFilters);
}
```

5. Add getter for outputsVersion:
```java
public long getOutputsVersion() {
    return outputsVersion.get();
}
```

6. Update createTaskStatus() to pass outputsVersion.get() to TaskStatus constructor.

Import ConcurrentMap, ConcurrentHashMap, TupleDomain, ImmutableMap as needed.

Add HTTP endpoint to fetch dynamic filters from workers.

1. In TaskManager interface (presto-main-base), add method:
```java
Map<String, TupleDomain<String>> getDynamicFilters(TaskId taskId, long sinceVersion);
```

2. In SqlTaskManager (presto-main-base), implement the method:
```java
@Override
public Map<String, TupleDomain<String>> getDynamicFilters(TaskId taskId, long sinceVersion) {
    return getTask(taskId, ImmutableList.of()).getDynamicFiltersSince(sinceVersion);
}
```

3. In TaskResource (presto-main), add endpoint:
```java
@GET
@Path("{taskId}/outputs/filter/{version}")
@Produces(APPLICATION_JSON)
public Response getDynamicFilters(
        @PathParam("taskId") TaskId taskId,
        @PathParam("version") long sinceVersion)
{
    requireNonNull(taskId, "taskId is null");
    Map<String, TupleDomain<String>> filters = taskManager.getDynamicFilters(taskId, sinceVersion);
    return Response.ok().entity(filters).build();
}
```

The endpoint returns empty map {} when no new filters since requested version, or map of filterId -> TupleDomain JSON when filters are available. TupleDomain already has Jackson serialization via existing infrastructure.

Import TupleDomain from com.facebook.presto.common.predicate.TupleDomain.
