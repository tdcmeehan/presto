  <name>Task 2: Add partition predicate matching utility in IcebergUtil</name>
  <files>presto-iceberg/src/main/java/com/facebook/presto/iceberg/IcebergUtil.java</files>
  <action>
Create a new method to check if a split's partition values match a filter predicate, modeled after metadataColumnsMatchPredicates():

1. Add method signature:
   ```java
   public static boolean partitionMatchesPredicate(
           TupleDomain<IcebergColumnHandle> filterConstraint,
           Map<Integer, HivePartitionKey> partitionKeys,
           PartitionSpec spec)
   ```

2. Early returns:
   - If filterConstraint.isAll() return true (no filtering)
   - If filterConstraint.isNone() return false (filter everything)
   - If partitionKeys is empty return true (unpartitioned table)

3. For each domain in filterConstraint:
   - Get the IcebergColumnHandle from the domain entry
   - Get the column ID from columnHandle.getId()
   - Look up the partition key by column ID in partitionKeys map
   - If partition key not found, skip this column (not a partition column)
   - If partition key found:
     a. Get the HivePartitionKey value (Optional<String>)
     b. If value is empty (null partition), check domain.includesNullableValue(null)
     c. If value present, convert string to appropriate Java type based on columnHandle.getType()
     d. Use domain.includesNullableValue(convertedValue) to check match
   - If any domain doesn't match, return false

4. Type conversion helper (inline or private method):
   - For VARCHAR: use Slices.utf8Slice(stringValue)
   - For BIGINT/INTEGER/SMALLINT/TINYINT: use Long.parseLong(stringValue)
   - For DOUBLE: use Double.parseDouble(stringValue)
   - For REAL: use Float.floatToIntBits(Float.parseFloat(stringValue)) cast to long
   - For DATE: use LocalDate.parse(stringValue).toEpochDay() as long
   - For BOOLEAN: use Boolean.parseBoolean(stringValue)
   - For DECIMAL: parse appropriately based on precision/scale
   - Default: log warning and return true (don't filter if unsure)

5. Return true if all domains match
